# Planck

## Descripcion y specs

este es mi teclado favorito, para llevar a cualquier lado, debido a su tama√±o es ideal
para llevar en la mochila junto a mi notebook.

Actualmente lo llevo con unos switches `DUROCK dolphins (creo que son de 62g)`, son
unos switches lineales silenciosos, ideales para cuando hay que ir a la oficina.
Adicional a eso lo uso con unas keycap `DROP + Matt3o MT3 Susuwatari - MT3 profile`,
el perfil mas unas patas de silicona, permiten tener un angulo levemente inclinado
lo cual es como a mis gigantes manos.

## layout

de momento he usado la configuracion por defecto que tiene el planck, lo he probado usado configurando,
las teclas especiales, con layouts distintos (4x12, doble mini barra espaciadora, etc.) y me siento
mas comodo usando esta configuracion, a lo mejor talvez remover los layouts que no utilizo (dvorak, colemak, plover),
adicional de remover cosas como la configuracion de encoder y dip switches, para dejar el codigo mas limpio.

(nota del autor: creo que es el teclado mas default que tengo, pero a la vez el que mas me gusta en este estado).
